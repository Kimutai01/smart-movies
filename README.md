
# SMART MOVIES

A website that enables a user to search a movie of his/her interest






## Run Locally

Clone the project

```bash
  git clone git@github.com:Kimutai01/smart-movies.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
